### 进程管理

**ps：** 查看进程 		-**ef**  显示系统中所有进程的全面信息			**aux**  显示所有用户有关进程的所有信息

```shell
#查看系统全部的进程
ps -ef
#显示所有用户有关进程的所有信息
ps -aux
```



**top：**动态显示进程的过程

```shell
#动态查看当前进程信息
# -c 列出完整指令信息
top -c
```



**kill：**终止进程		**-9** 强制终止进程

```shell
#强制停止掉进程id为123的进程
kill -9 123
```
